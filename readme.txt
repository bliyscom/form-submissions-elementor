A custom Elementor widget to display form submissions in a table on the frontend.

== Description ==

This plugin provides a custom Elementor widget that allows you to display submissions from Elementor forms in a customizable table. You can select the form, choose fields to display, and apply styling options including pagination and items per page.

Features:
- Select any Elementor form
- Choose specific fields or display all
- Styling controls for table appearance
- Pagination with items per page selector (up to 100)

Requires Elementor and Elementor Pro (for forms).

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/elementor-form-submissions` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. In Elementor editor, search for 'Form Submissions Table' widget and add it to your page.
4. Configure the widget settings as needed.

== Frequently Asked Questions ==

= Does this require Elementor Pro? =

Yes, as it relies on Elementor forms which are part of Elementor Pro.

= How do I select which form to display? =

In the widget controls, there's a dropdown to select from available forms.
